# 💜 TransactIQ: Digital Payment Insights Platform

## 📌 Project Overview
TransactIQ is a data-driven analytics platform designed to understand digital payment behavior across India. The project analyzes transaction data from PhonePe to uncover patterns in user behavior, geographical trends, and payment preferences.

---

## 🎯 Objective
To analyze and visualize digital payment data in order to:
- Understand user behavior and engagement
- Identify top-performing regions
- Analyze transaction trends over time
- Provide actionable business insights

---

## 🛠️ Tech Stack
- Python (Pandas, Matplotlib, Plotly)
- SQL (MySQL)
- Streamlit (Dashboard)
- VS Code

---

## 🧱 Data Pipeline
1. Extracted raw JSON data from PhonePe dataset  
2. Transformed and loaded data into MySQL database  
3. Performed SQL-based analysis  
4. Visualized insights using Python  
5. Built interactive dashboard using Streamlit  

---

## 📊 Key Insights

### 📍 Regional Insights
- Telangana leads in total transaction value  
- Maharashtra has highest transaction count  
→ Indicates higher frequency but lower average transaction size  

---

### 💳 Payment Behavior
- Merchant payments dominate transaction types  
→ Suggests strong real-world usage of digital payments  

---

### 📈 Growth Trends
- Digital payments show overall upward growth  
- Significant spike observed in recent years  
→ Reflects increasing adoption of digital platforms  

---

### 🔄 Seasonal Trends
- Transactions increase consistently from Q1 to Q4  
- Peak observed in Q4  
→ Indicates festive-driven spending behavior  

---

## 💡 Business Implications
- High-value regions can be targeted for premium financial services  
- High-frequency regions can be targeted for engagement strategies  
- Seasonal spikes can guide marketing campaigns  
- Merchant ecosystem is key for expansion  

---

## 📊 Dashboard Features
- Interactive filters (State, Year)  
- KPI metrics (Total Transaction Value, Top State)  
- Visualizations:
  - Yearly trend
  - Quarterly trend
  - Top states
  - Transaction type distribution  

---

## 🚀 Conclusion
TransactIQ successfully transforms raw transaction data into meaningful insights through a structured pipeline and interactive dashboard. The project demonstrates the integration of data engineering, analytics, and design thinking.
