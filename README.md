# 💜 TransactIQ

A data-driven analytics platform to understand digital payment behavior across India.

## 🚀 Features
- SQL-based data analysis
- Python visualizations
- Interactive Streamlit dashboard
- Regional and behavioral insights

## 🛠️ Tech Stack
- Python
- MySQL
- Streamlit
- Pandas, Plotly

## 📊 Key Insights
- Telangana leads in transaction value
- Maharashtra dominates in transaction count
- Merchant payments are most used
- Strong seasonal trends observed
